CARPETA: front/ (Frontend - Interfaz de Usuario)
---------------------------------------------------

Archivos:
  • index.html             - Página principal con hero, búsqueda y grid
  • auth.html              - Página de login/registro
  • app.js                 - Lógica de JavaScript (búsqueda, filtrado)
  • styles.css             - Diseño CSS moderno y responsivo
  • README.md              - Documentación del frontend

Para ejecutar:
  cd front
  python3 -m http.server 8000
  
  Luego abre: http://localhost:8000


CARPETA: back/ (Backend - Servidor y API)
------------------------------------------

Archivos principales:
  • server.js              - Servidor Express
  • package.json           - Dependencias npm
  • data.js                - Base de datos en memoria (eventos)
  • config.js              - Configuración centralizada
  • start.sh               - Script para iniciar
  • .env.example           - Variables de entorno

Subcarpeta routes/ (API Endpoints):
  • events.js              - 5 endpoints para eventos
  • users.js               - 4 endpoints para usuarios
  • orders.js              - 4 endpoints para órdenes/compras

Para ejecutar:
  cd back
  npm install    (solo primera vez)
  npm start
  
  Servidor en: http://localhost:3000/api


INSTRUCCIONES DE INICIO
------------------------

1. Terminal 1 - Backend:
   $ cd /path/to/FESTIVALESV/back
   $ npm install
   $ npm start
   
   Resultado: Servidor en http://localhost:3000

2. Terminal 2 - Frontend:
   $ cd /path/to/FESTIVALESV/front
   $ python3 -m http.server 8000
   
   Resultado: Sitio en http://localhost:8000

3. Abre tu navegador:
   http://localhost:8000
